## Description
This is the Linux (and maybe MacOS, to some extent) environment I work with daily. Please refer to each subfolder's `README.md` file for instructions on how to install, setup and/or tweak the indicated environment as you please.

### Disclaimers
Please keep the following in mind:

-   _My `bash` environments are extremely specific and require many personal/third-party scripting to keep stable, so be careful if, for any reason, you wish to either use them or import pieces of them._
-   _For the time being, my instructions are catered to **Arch-based distros** (more specifically, Manjaro **XFCE**), so some tweaks I've included may not apply to your environment. Please consult online instructions for your distro and/or contact me if you'd like assistance or instructions for other distros to be included._
