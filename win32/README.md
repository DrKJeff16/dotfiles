## Description
This is the Windows environment I work with daily. Please refer to each subfolder's `README.md` file for instructions on how to install, setup and/or tweak the indicated environment as you please.

### Disclaimers
Please keep the following in mind:

-   _My `bash` environments are extremely specific and require many personal/third-party scripting to keep stable, so be careful if, for any reason, you wish to either use them or import pieces of them._
-   _I generally use [Git Bash](https://gitforwindows.org) as my workspace, but I may have other configurations specifically for a [MSYS2](https://www.msys2.org) environment._
