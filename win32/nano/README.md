## Description
This is the folder containing my `nanorc` file and various other tweaks for `GNU Nano`.

## Dependencies
My nano environment is quite simplistic at the moment, since I don't use it that much. I only use an imported [syntax highlighting repository](https://github.com/scopatz/nanorc) made by [scopatz](https://github.com/scopatz), with added indent/line-numbers.

## Installation
If you are in a Windows 10 environment, the files included in this subfolder should be exported as such:

-   `nanorc -> ~/.nanorc`
-   `.nano -> ~/.nano`

With `~` being your `%USERPROFILE%` directory (or `C:\Users\<your_user>`).
