## Description
This is the folder containing my `vimrc` file and various other tweaks for Vim.

## Dependencies
My vim environment uses a lot of plugins and custom/recommended tweaks to them to make my workflow much more dynamic and efficient.

<h3 id="plugin-manager"><a href="https://github.com/VundleVim/Vundle.vim">Vundle.vim</a></h3>
I use `Vundle.vim` as my default plugin manager. Installation instructions can be found in [the repository](https://github.com/VundleVim/Vundle.vim).

## Installation
If you are in a Windows 10 environment, the files included in this subfolder should be exported as such:

-   `vimrc -> ~/_vimrc`
-   `vimfiles -> ~/vimfiles`

With `~` being your `%USERPROFILE%` directory (or `C:\Users\<your_user>`).
