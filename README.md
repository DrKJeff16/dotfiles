## Description
A collection of my dotfiles, configuration files and other file structures for various programs I use daily.

### Structure

This repository has two main directories: `unix` _for my Linux environments_ and `win32` _for my Windows environments_. Each subdirectory has its own `README` file detailing their dependencies, structure and more.

A very general structure of this repository would look as follows:

```
dotfiles/
|____ unix/
     |____ bash/
		 |____ nano/
		 |____ nvim/
		 |____ README.md
		 |____ vim/
		 ...
|____ win32/
     |____ mintty/
     |____ msys2/
	   |____ nano/
		 |____ README.md
		 |____ vim/
		 ...
|____ LICENSE
|____ README.md
...
```
